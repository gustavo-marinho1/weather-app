
## Wweather App
Weather app was developed by Gustavo Marinho. You can check the weather in any city.


## Technology

* Typescript
* React
* Vite.js
* Redux
* Axios

## Services Used

* Github
* API: https://openweathermap.org/api


## Getting started

* To install the dependencies:
>    npm install

* To run the project:
>    npm run dev
